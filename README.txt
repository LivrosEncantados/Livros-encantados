
# Livros Encantados - Site

## Instruções

1. Vá para https://github.com e crie um repositório público com nome `livros-encantados`.
2. Envie os arquivos deste .zip para o repositório.
3. Vá até as configurações do repositório > Pages.
4. Ative o GitHub Pages selecionando a branch `main` e a pasta `/ (root)`.
5. Acesse seu site em:
   https://SEU-USUARIO.github.io/livros-encantados/
   (troque SEU-USUARIO pelo seu nome de usuário do GitHub)
